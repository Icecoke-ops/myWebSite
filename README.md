## 关于这个项目

这个项目是我的个人博客的前端页面的部分，其中所有的内容部分来自 [api.icecoke.cn](https://api.icecoke.cn)，后台的所有的接口都是配置的可跨域的的，之所以这样设计就是为了能够方便进行调试、部署。其实后端程序也很简单，就是一个文件的托管服务，目前时在阿里云上的，后期为了进一步压缩成本可能会迁移到其它的平台。

前端页面使用了 Cloudflare 的 workers 进行自动化的部署。你可以通过 [https://icecoke.cn](https://icecoke.cn) 进行访问。

本项目基于 MIT 开源，如果你看得上，尽管拿去用吧！！
